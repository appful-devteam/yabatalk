主な機能

トーク解析

LINEのトーク履歴を解析し、会話のパターンを可視化します。

関係性診断

ふたりの関係性タイプを診断し、相性のポイントをお伝えします。

メッセージ数や返信速度など、詳細な統計を確認できます。

プライバシーポリシー

最終更新日: 2026年5月18日

めろとーく（以下「本アプリ」）は、株式会社appful（以下「運営者」）が提供します。運営者は、ユーザーのプライバシーを尊重し、個人情報の保護に努めています。本プライバシーポリシーは、本アプリが取り扱う情報の種類、利用目的、第三者への提供、およびユーザーの権利について説明します。本アプリのご利用をもって、本プライバシーポリシーに同意したものとみなします。

1. 収集する情報

本アプリは以下の情報を取り扱います：

  • トーク履歴データ: ユーザーがインポートしたLINEのトーク履歴ファイル（テキスト形式）を解析のために使用します。メッセージ本文、送信者名、送信日時が含まれます。
  • 解析結果データ: トーク履歴から算出された4軸スコア（トーク量・感情温度・リズム・深度）、統計情報、性格タイプ分類、診断結果。
  • サブスクリプション情報: Apple App Store 経由での Premium／Premium+ プランの購入履歴。本アプリが決済情報を直接収集することはありません。
  • 匿名利用統計・エラーログ: アプリの機能改善および障害対応のため、匿名化された利用統計情報・エラーログを Google LLC が提供する Firebase（Firebase Analytics・Crashlytics）に送信する場合があります。個人を特定する情報は含まれません。
  • 認証用識別子・プッシュ通知トークン: コミュニティ機能・通知機能の提供のため、Firebase Authentication が発行する匿名 UID および APNs プッシュトークンを Firebase（Firestore／Cloud Functions）上に保存します。

2. データの保存場所と管理

  • ローカル保存（原則）: トーク履歴データおよび解析結果は、原則としてユーザーのデバイス内にのみ保存されます（Apple SwiftData フレームワークを使用）。
  • 外部送信が発生する場合: AIサマリー機能・返信提案（相談）機能・擬人化チャット機能を利用する場合に限り、トーク履歴の一部が Google LLC のサーバー（Gemini API）に送信されます。
  • データの保持期間: デバイス内のデータは、ユーザーがアプリを削除するか、設定画面の「すべてのデータを削除」を実行するまで保持されます。
  • AI機能を利用しない場合、トーク履歴が外部に送信されることは一切ありません。

3. 情報の利用目的

収集した情報は以下の目的にのみ利用します：

  • トーク履歴の解析および4軸スコア・診断結果の生成
  • AI機能（サマリー生成・返信提案・擬人化チャット）の提供
  • アプリ機能の提供、維持、改善
  • エラーの検出および修正
  • ユーザーサポートへの対応

4. 第三者への提供

本アプリは、以下の場合を除き、ユーザーの個人情報を第三者に提供することはありません：

  • ユーザーが明示的に同意した場合（AI機能の初回利用時の同意画面）
  • 法令に基づく開示要請があった場合

本アプリが利用する外部サービス：

  • Apple App Store（アプリ配信およびサブスクリプション決済）
  • Google Gemini API（AI機能） — 運営: Google LLC
  • Firebase（Analytics・Crashlytics・Cloud Functions・Firestore・Authentication・Remote Config・Cloud Messaging） — 運営: Google LLC
  • Google AdMob（広告配信） — 運営: Google LLC

5. Google Gemini API の利用について

AIサマリー機能、返信提案（相談）機能、および擬人化チャット機能では、Google LLC が提供する Gemini API を利用しています。送信されるデータはメッセージ本文・送信者名・送信日時に限られ、画像・動画・スタンプ等のメディアデータは送信されません。Google による取り扱いの詳細は、Google のプライバシーポリシーおよび Gemini API 利用規約をご参照ください。

6. トーク相手のプライバシーについて

トーク履歴にはユーザー以外の方（トーク相手）の発言や個人情報が含まれています。日本の個人情報保護法に基づき、第三者の個人情報を外部サービスに提供する際は、原則としてその方の同意が必要です。ユーザーは、本アプリの利用にあたり、トーク相手のプライバシーに十分配慮し、必要に応じて同意を得る等の適切な措置を講じてください。

7. セキュリティ

本アプリは、ユーザーデータの保護のため、Apple SwiftData によるローカル管理、HTTPS による暗号化通信、API キーのサーバー側管理などのセキュリティ対策を講じています。ただし、インターネット上のデータ送信において、完全なセキュリティを保証することはできません。

9. ユーザーの権利

  • データの削除: 設定画面の「すべてのデータを削除」より、保存されたすべてのデータを削除できます。
  • AI機能の不使用: AI機能を利用しないことで、外部へのデータ送信を完全に回避できます。
  • アプリの削除: アプリを削除することで、デバイス内のすべてのデータが削除されます。
